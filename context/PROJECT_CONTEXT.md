# Project Context: Alumni Platform

## Goal
A platform to manage and track alumni data, including scraping features to keep information up to date.

## Tech Stack
- **Frontend**: Vue 3, Vite, Bun, TanStack Table/Query, Shadcn-vue.
- **Backend**: Fastify, Bun, MongoDB (Mongoose), Zod.
- **Tools**: Playwright for scraping.
