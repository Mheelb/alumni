# AI Coding Guidelines

- **Architecture**: Feature-oriented structure in `apps/web/src/features`.
- **Backend**: Fastify with TypeScript and Mongoose.
- **Validation**: Use Zod for all input validation and schema definitions.
- **UI**: Shadcn-vue components and Tailwind CSS.
- **State Management**: TanStack Query for server state.
- **Scraping**: Playwright for all data extraction tasks.
